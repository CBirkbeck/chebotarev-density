# Chebotarev density theorem

A Lean 4 / Mathlib formalisation of the Chebotarev density theorem in
conjugacy-class form for a finite Galois extension of number fields.

## Status

Skeleton + decomposition. All theorem proofs are `sorry`; every
definition has real content (`MulAction.stabilizer`, `Ideal.inertia`,
`Classical.choose` of existence theorems). Build clean: `lake build`
returns success with `sorry` warnings only.

See `blueprint/print/print.pdf` for the human-readable outline and
`blueprint/web/index.html` for the dep-graph.

## References

Both included in `docs/`:

* P. Stevenhagen, H.W. Lenstra, *Chebotarëv and his density theorem*
  (`docs/cheb.pdf`).
* R. Sharifi, *Algebraic Number Theory* — §7.1–7.2 (`docs/algnum.pdf`).

The proof follows the modern exposition of these references: the
cyclotomic case via Dirichlet's `ζ_L = ∏_χ L(χ,·)` factorisation, the
abelian case via Chebotarev's compositum trick, and the general Galois
case via reduction to the cyclic subextension generated by a class
representative.
