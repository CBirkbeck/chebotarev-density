## The clean statement to formalize

Let K be a number field of degree d, 𝔮 a nonzero integral ideal. Let I(𝔮) be the group of
fractional ideals prime to 𝔮, P_𝔮 the subgroup generated by principal ideals (α) with α ≡ 1
(mod 𝔮) (and, narrow version, positivity at real places). Put Cl_𝔮 = I(𝔮)/P_𝔮.

For every ray class A ∈ Cl_𝔮:

  #{ 𝔞 ⊆ 𝓞_K : (𝔞,𝔮)=1, [𝔞]_𝔮 = A, N𝔞 ≤ X } = ρ_𝔮·X + O_{K,𝔮}(X^{1−1/d}),

with **ρ_𝔮 independent of A**.

- **Lang, GTM 110, Chapter VI §3, Theorem 3:** for a cycle 𝔠 and generalized ideal class 𝔎 of
  I(𝔠)/P_𝔠, j(𝔎,t) = ρ_𝔠·t + O(t^{1−1/N}), ρ_𝔠 independent of 𝔎. Ordinary ideal-class case
  immediately below with constant 2^{r₁}(2π)^{r₂}R/(w√|d_K|). Proved from the lattice-point
  theorem in Chapter V §2 and the unit fundamental-domain construction in Chapter VI §3.
- **Modern explicit reference:** Gun–Ramaré–Sivaraman, "Counting ideals in ray classes," JNT 243
  (2023), Theorem 1. Narrow ray class group H_𝔮(K) = I(𝔮)/P_𝔮; uniform-in-class main term ∝ x +
  x^{1−1/n_K} error, constants displayed. Cites Lang GTM 110; uses a Widmer lattice-point theorem
  for bounded sets with Lipschitz boundary.

## How it gives Sharifi's missing value-fibre estimate

ψ : Cl_𝔮 → μ_n nontrivial finite-order character, ψ(𝔞) := ψ([𝔞]_𝔮) for 𝔞 prime to 𝔮, 0 else.
S_ψ(X) = Σ_{N𝔞≤X} ψ(𝔞) = Σ_{A∈Cl_𝔮} ψ(A)(ρ_𝔮 X + O(X^{1−1/d})). Since ψ nontrivial,
Σ_A ψ(A) = 0, so S_ψ(X) = O(X^{1−1/d}). For value-fibres: A_ψ(X,ζ) = |ker ψ|·ρ_𝔮·X +
O(X^{1−1/d}) for ζ in image of ψ; if ord ψ = n, image = all μ_n, uniform in every n-th root of
unity. Exactly the C-independent-of-ζ statement.

## The geometry-of-numbers proof, formalization-ready

**Core lattice lemma.** Λ ⊂ ℝ^d a full lattice, D ⊂ ℝ^d bounded with boundary covered by
finitely many Lipschitz maps [0,1]^{d−1} → ℝ^d. Then, uniformly for translates v+Λ:

  #((v+Λ) ∩ tD) = (vol D / covol Λ)·t^d + O_{Λ,D}(t^{d−1}).

Lang: compare lattice fundamental-parallelepiped cells inside tD with those meeting the boundary;
the boundary count is O(t^{d−1}) (partition each Lipschitz chart domain into ≍ t^{d−1} small
cubes; each image piece has bounded diameter after dilation; a bounded set meets only O_Λ(1)
lattice cells). **Lang page 129** gives exactly this, ending with b(t) ≤ C'[t]^{N−1}. Take
t = X^{1/d}: main term t^d → X, boundary error t^{d−1} → X^{(d−1)/d} = X^{1−1/d}.

For the ideal/ray-class count: reduce ideals in a fixed generalized class to lattice points in a
translate of an ideal lattice inside a unit fundamental domain D; congruence conditions become a
lattice translate; covolume is the same for every translate; the unit fundamental domain has
Lipschitz-parametrizable boundary.

**Exact boundary regularity to formalize: finite Lipschitz parametrizability of ∂D.**
(Semialgebraic / C¹-piecewise would be more than enough; Lang only needs Lipschitz charts.)
Exponent 1−1/d is the natural "easy" geometry-of-numbers boundary exponent; weaker O(X^{1−δ})
suffices for the Abel-summation step but 1−1/d is the clean target.

## The class-field-theory subtlety (CORRECTION)

§7 is essentially right but do **NOT** formalize as "all norm residue classes mod m have the same
constant." **That is FALSE unless restricted to the image of the norm map.**

Counterexample: K = ℚ(i), ideals prime to 2: odd ideal norms are always ≡ 1 (mod 4), not
equidistributed between 1 and 3 (mod 4). Correct statement: **ray classes are equidistributed**;
norm-residue fibres are unions of ray classes. Fibres in the image of the norm map have equal
numbers of ray classes; residue classes outside the image have zero count.

For K(μ_m)/K, no CFT: if 𝔭 ∤ m, Frob_𝔭(ζ_m) = ζ_m^{N𝔭}, so Frobenius depends only on N𝔭 mod m.
Define θ(𝔞): ζ_m ↦ ζ_m^{N𝔞} (multiplicative). If 𝔞 = (α), α ≡ 1 (mod m𝓞_K), then N_{K/ℚ}(α) ≡
1 (mod m), so θ(𝔞) = 1. Hence θ factors through a ray class group — **elementary** (uses the
determinant interpretation of the norm and the Frobenius action on roots of unity, NOT Artin
reciprocity). For general abelian L/K, χ∘Frob factoring through a ray class group IS the
conductor/Artin-reciprocity content of CFT (S–L say so explicitly); Chebotarev's crossing avoids it.

## Best route for Q4 — formalize route (i)

1. Prove the ray-class count with class-independent main term.
2. Deduce nontrivial ray-class character sums are O(X^{1−1/d}).
3. Specialize cyclotomic ideal characters to ray-class characters by the elementary norm/Frobenius
   argument.

Cleaner than total-count-then-equidistribute (which is false outside the norm image or silently
reintroduces the same ray-class structure).

**Actionable replacement for the blocker:**
  ∀ A ∈ Cl_𝔮, #{𝔞 : (𝔞,𝔮)=1, [𝔞]_𝔮=A, N𝔞 ≤ X} = ρ_𝔮 X + O(X^{1−1/d}), ρ_𝔮 independent of A.
From that, the Chebotarev cyclotomic L-function continuation follows by one line of finite-character
orthogonality.
