## Sources

- **Sharifi** = *Algebraic Number Theory* (`docs/algnum.pdf`).

The Frobenius-symbol-as-conjugacy-class facts are SL pp. 13–14:
> (b) "if ψ, ψ′ are two places over p, then ψ′ = ψ ∘ τ for some τ ∈ G"
> (p. 14) "Frob_{ψ∘τ} = τ⁻¹ ∘ Frob_ψ ∘ τ; that is, if ψ varies over the places
> over a fixed prime p, then Frob_ψ ranges over a conjugacy class in G … it is
> well-defined only up to conjugacy, and it is called the Frobenius substitution of p."

Places over `p` ↔ primes `𝔓 | 𝔭`; `ψ ↦ ψ ∘ τ` ↔ the Galois action `𝔓 ↦ σ • 𝔓`.

## R: `exists_frobeniusClass` — DONE (composed sorry-free this pass)

`Frobenius.lean`. The Frobenius conjugacy class of an unramified `𝔭` is well-defined:
there is a `C : ConjClasses Gal(L/K)` equal to `mk (frobeniusAt 𝔓)` for *every* prime
`𝔓 | 𝔭`. SL p. 14: the Frobenius substitution is the conjugacy class swept out as `𝔓`
ranges over primes above `𝔭`.

Composition (`exists_frobeniusClass := by …`, sorry-free):
- pick `𝔓₀ | 𝔭` via **L-new** `exists_prime_liesOver`;
- set `C := mk (frobeniusAt 𝔓₀)`;
- for any `𝔓 | 𝔭`, `mk (frobeniusAt 𝔓₀) = mk (frobeniusAt 𝔓)` by
  `ConjClasses.mk_eq_mk_iff_isConj` + **L1** `frobeniusAt_isConj_of_liesOver`.

### L-new — `exists_prime_liesOver` (leaf, PROVEN from mathlib)
A nonzero prime `𝔭` of `𝓞 K` has a prime `𝔓 | 𝔭` of `𝓞 L`, and any such `𝔓 ≠ ⊥`.
- Discharge (verified, `lean_run_code` + `lake build`):
  `Ideal.exists_ideal_over_prime_of_isIntegral_of_isDomain` (going-up, integral
  extension `𝓞 K ⊆ 𝓞 L`) gives the prime with `comap = 𝔭`; nonzero from
  `RingHom.ker_eq_comap_bot` + injectivity of `algebraMap (𝓞 K) (𝓞 L)`
  (`FaithfulSMul.algebraMap_injective`).
- Attacks: [1] counterexample — going-up is unconditional for integral extensions,
  no contradiction. [2] edge `𝔭 = ⊥` excluded by hypothesis; `𝔓 = ⊥` ⇒ `𝔭 = ⊥` via
  the kernel argument, refuted. [3] hypotheses: `hpr` needed (going-up wants a prime),
  `hnz` needed for the nonzero conclusion. Survived.

### L1 — `frobeniusAt_isConj_of_liesOver` — DONE (composed sorry-free this pass)
Two Frobenius elements above the same unramified `𝔭` are conjugate. Composed from:
  - **transitivity** (SL fact b): `∃ σ : Gal(L/K), σ • 𝔓 = 𝔓'`, discharged by mathlib
    `Ideal.exists_smul_eq_of_isGaloisGroup 𝔭 𝔓 𝔓' Gal(L/K)`
    (`Mathlib.NumberTheory.RamificationInertia.Galois`) — all instances
    (`IsGaloisGroup Gal(L/K) (𝓞 K) (𝓞 L)`, `Finite`, the `LiesOver`/`IsPrime` ones)
    resolved automatically; verified by build.
  - **L1.1 (new leaf, `sorry`)** `frobeniusAt_conj_eq`: `σ • 𝔓 = 𝔓' →
    frobeniusAt 𝔓' = σ · frobeniusAt 𝔓 · σ⁻¹` — SL p. 14 `Frob_{ψ∘τ} = τ⁻¹ Frob_ψ τ`.
    Substantive: `σ · Frob_𝔓 · σ⁻¹` satisfies the `frobeniusAt` spec at `𝔓'` (stabilises
    `𝔓'`; acts as `N𝔭`-power on `κ(𝔓')`), then `exists_unique_frobeniusAt` uniqueness.
  - parent body: `obtain ⟨σ, hσ⟩ := exists_smul_eq_of_isGaloisGroup …;
    rw [frobeniusAt_conj_eq … σ hσ]; exact isConj_iff.mpr ⟨σ, rfl⟩`.
