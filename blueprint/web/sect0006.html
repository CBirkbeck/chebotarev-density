1 Chebotarev density theorem

The general Galois case is reduced to the abelian case via the cyclic subextension generated by a class representative. Source: Sharifi 7.2.2 Step 1 (p. 143).

Lemma 1.1

Lean declarations

  • Chebotarev.count_primes_above_with_frobenius_eq_sigma

Let \(L/K\) be a finite Galois extension with \(G=\mathrm{Gal}(L/K)\), \(C\subseteq G\) a conjugacy class, \(\sigma \in C\), and \(\mathfrak {p}\) a nonzero prime of \(\mathcal{O}_K\) unramified in \(L\) with \(\sigma _\mathfrak {p}= C\). The number of primes \(\mathfrak {P}\) of \(\mathcal{O}_L\) above \(\mathfrak {p}\) with \(\mathrm{Frob}_\mathfrak {P}= \sigma \) is exactly \(\left\lvert G \right\rvert /(f\cdot \left\lvert C \right\rvert )\), where \(f = \mathrm{ord}(\sigma )\).

Proof ▶

The Galois group \(G\) acts transitively on primes above \(\mathfrak {p}\), with stabiliser \(D_\mathfrak {P}\). Hence the orbit has size \(\left\lvert G \right\rvert /\left\lvert D_\mathfrak {P} \right\rvert \). For unramified \(\mathfrak {p}\), \(D_\mathfrak {P}=\left\langle \mathrm{Frob}_\mathfrak {P} \right\rangle \) has order \(f\), so there are \(\left\lvert G \right\rvert /f\) primes above \(\mathfrak {p}\). Their Frobenius elements form a \(G\)-orbit in \(C\) (under conjugation), which is all of \(C\) by transitivity; this orbit has size \(\left\lvert C \right\rvert \), and within each orbit the count of representatives \(\sigma \) is \(\left\lvert G \right\rvert /(f\cdot \left\lvert C \right\rvert )\) (Sharifi p. 143).

Lean declarations

  • Chebotarev.density_lift_through_fixedField

Let \(\sigma \in \mathrm{Gal}(L/K)\), \(E = L^{\left\langle \sigma \right\rangle }\), and \(\sigma _E\in \mathrm{Gal}(L/E)\) the corresponding element. Given the abelian-case density over \(E\) for the Frobenius-fibre of \(\sigma _E\) (value \(1/\left\lvert \mathrm{Gal}(L/E) \right\rvert \)), the \(K\)-density of the Frobenius class \([\sigma ]\) is \(\left\lvert [\sigma ] \right\rvert /\left\lvert G \right\rvert \).

Proof ▶

For primes \(P\) of \(E\) above \(\mathfrak {p}\in S = \left\{ \mathfrak {p} \; \middle |\; \sigma _\mathfrak {p}= [\sigma ] \right\} \), exactly \(\left\lvert G \right\rvert /(f\left\lvert C \right\rvert )\) of the primes of \(L\) above have Frobenius \(\sigma \) (lemma 1.1); with \(\sum _\mathfrak {p}N\mathfrak {p}^{-s} \sim \sum _P NP^{-s}\) (lemma 1.10 over both \(K\) and \(E\)), \(\delta _K(S) = (f\left\lvert C \right\rvert /\left\lvert G \right\rvert )\, \delta _E(T_\sigma ) = (f\left\lvert C \right\rvert /\left\lvert G \right\rvert )(1/f) = \left\lvert C \right\rvert /\left\lvert G \right\rvert \).

Lean declarations

  • Chebotarev.chebotarev_density

For a finite Galois extension \(L/K\) of number fields with \(G=\mathrm{Gal}(L/K)\) and any conjugacy class \(C\subseteq G\),

\[ \delta \bigl(\left\{ \mathfrak {p}\subset \mathcal{O}_K \; \middle |\; \sigma _\mathfrak {p}= C \right\} \bigr) \; =\; \frac{\left\lvert C \right\rvert }{\left\lvert G \right\rvert }. \]
Proof ▶

Pick a representative \(\sigma \in C\) (ConjClasses.mk_surjective) and let \(E = L^{\left\langle \sigma \right\rangle }\) be the fixed field of \(\left\langle \sigma \right\rangle \). The map \(\left\langle \sigma \right\rangle \xrightarrow {\sim } \mathrm{Gal}(L/E)\) (IntermediateField.subgroupEquivAlgEquiv) shows \(\mathrm{Gal}(L/E)\) is commutative (as \(\left\langle \sigma \right\rangle \) is cyclic), so \(L/E\) is abelian and theorem 1.8 applies to its Frobenius \(\sigma _E\), giving \(\delta _E(T_\sigma ) = 1/\left\lvert \mathrm{Gal}(L/E) \right\rvert \). Feeding this into the density lift lemma 1.2 yields \(\left\lvert C \right\rvert /\left\lvert G \right\rvert \).

Lean declarations

  • Chebotarev.chebotarev_density_of_comm

When \(G = \mathrm{Gal}(L/K)\) is abelian, the Chebotarev density theorem holds: for every conjugacy class \(C\), \(\delta (\left\{ \mathfrak {p} \; \middle |\; \sigma _\mathfrak {p}= C \right\} ) = \left\lvert C \right\rvert /\left\lvert G \right\rvert \).

Proof ▶

In an abelian group each class \(C = [\sigma ]\) is a singleton (lemma 1.6), so \(\left\lvert C \right\rvert = 1\) and the target density is \(1/\left\lvert G \right\rvert \); this is exactly theorem 1.8 for the representative \(\sigma \).

Lean declarations

  • Chebotarev.density_split_completely

For a finite Galois extension \(L/K\) of number fields, the Dirichlet density of primes of \(\mathcal{O}_K\) that split completely in \(L\) equals \(1/[L:K]\).

Proof ▶

A prime \(\mathfrak {p}\) splits completely in \(L\) iff every \(\mathfrak {P}\mid \mathfrak {p}\) has \(\mathrm{Frob}_\mathfrak {P}= 1\), i.e. the Frobenius conjugacy class is the trivial class. Apply theorem 1.3 at \(C = \mathrm{Conj}(G)_{\{ 1\} }\); by lemma 1.5 the numerator \(\left\lvert C \right\rvert = 1\) and by lemma 1.7 the denominator \(\left\lvert G \right\rvert = [L:K]\), so the density equals \(1/[L:K]\).

Lean declarations

  • Chebotarev.dirichlet_primes_in_AP

For \(n \ge 1\) and \(a \in (\mathbb {Z}/n\mathbb {Z})^\times \), the Dirichlet density of the rational primes \(p\) with \(p \equiv a \pmod{n}\) equals \(1/\varphi (n)\).

Proof ▶

Specialise theorem 1.3 to \(K = \mathbb {Q}\), \(L = \mathbb {Q}(\zeta _n)\). The cyclotomic character identifies \(\mathrm{Gal}(\mathbb {Q}(\zeta _n)/\mathbb {Q})\) with \((\mathbb {Z}/n\mathbb {Z})^\times \); under this identification, the Frobenius \(\sigma _p\) at unramified \(p\) corresponds to \(p \bmod n\). The Galois group is abelian, so each conjugacy class is a singleton; the set of primes with \(p \equiv a \pmod{n}\) is the Frobenius preimage of \(\{ a\} \), whose density is \(1/\varphi (n)\).
